Using Urban Airship with Titanium SDK

NOTE: If you are using version 1.7.3 or newer of Titanium SDK then you do not need to perform the steps outlined in this document.

The Urban Airship module requires some additional initialization functionality that is not available in Titanium SDK 1.7.2 or earlier. An additional initialization step is required when the Titanium application is created. This additional step is located in the generated application java file. Since this file is generated every time that you build your project, it is necessary for you to create a persistent application java file that will be used each time that you build your project.

You will need to perform the following steps for the Urban Airship module to function properly.

Step 1: Locate the generated application java file

  • Build and run your application on either an Android device or emulator.
  • In your project's root directory, locate the 'build/android/gen' directory
  • Locate the Application.java file in the directory hierarchy. The directory hierarchy is based on your application ID. For example, if your application ID was 'com.mycompany.myapp', then the file would be called 'myappApplication.java' and would be located in the 'gen/com/mycompany/myapp' directory.

Step 2: Create platform directory

  • In your project's root directory, create a 'platform/android' directory
  • Create a directory hierarchy identical to that in step 1 under this new directory. For example, you would create a 'gen/com/mycompany/myapp' directory under the 'platform/android' directory.

Step 3: Create persistent application java file

  • Copy the generated Application.java file (from step 1) into the platform directory (from step 2)
  • Edit the Application.java file
  • Locate the 'onCreate' method
  • Add the following code at the end of the method: 
        ti.modules.titanium.urbanairship.UrbanAirshipModule.onAppCreate(this);
  • Save the file

Step 4: Perform a clean build of your application

  • Select "Project" and then "Clean.." from the Ti Studio menu
  • Select the application project and click "OK"
  • Rebuild the application
